FAQ

Fast answers to the common issues (including why web downloads sometimes fail).

Why does the web downloader sometimes not download?
Browsers often block cross-domain downloads (CORS) or don’t allow saving files returned from certain servers. That’s why the app is more reliable. On web, we fall back to opening in a new tab so you can save manually.
What does Unisaver do exactly?
Unisaver is an all-in-one downloader + media manager. It supports social platforms and 1000+ sites, status saving, no-watermark options where possible, batch/background downloads, ad-block browser, VPN, speed test, link store, vault/private folder, and more. :contentReference[oaicite:2]{index=2}
Can I host this site on GitHub and keep it free?
Yes. Put this file + your images in a GitHub repo and enable GitHub Pages. Or use Cloudflare Pages / Netlify for free static hosting.
How do I keep seller.json/config outside the HTML?
Put a config.json in your site root and load it on page start. (Steps are below this code.)
